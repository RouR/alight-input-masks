How to Install.

Import the alight_money.js script in your page

<script src="alight_money.js"></script>

How to use it

Basic usage

<input type="text" name="field14" al-money="defaultMoney">

Model value: {{defaultMoney}}

Extended usage

<input type="text" name="field14" al-money="custom1" 
	money-decimal-delimiter="/" money-thousands-delimiter=" " 
	money-currency="&:">

Model value: {{custom1}}
